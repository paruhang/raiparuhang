— About

A quiet, durable place for the small tools you reach for often.

Raiparuhang is a one-person project. It exists because the internet is loud, and most utility sites are full of ads, sign-ups, and trackers — for tasks that should take ten seconds.

What it is

A small archive of focused tools that run entirely inside your browser. There is no backend, no upload, no account. Open the page, do the thing, close the tab.

How it's built

Just HTML, CSS, and JavaScript. The site is static. Each tool is a single page that ships only the code it needs. That's why it loads fast and works offline once the page is cached.

What it isn't

It is not a SaaS. It is not collecting your data. It is not trying to grow. It is meant to be a steady, useful, low-noise reference you can bookmark.
