Why use this image compressor?

🗜️

Compress JPG, PNG & WebP

Works as a photo compressor, picture compressor and image size reducer in one. Supports JPG, JPEG, PNG, WebP, BMP and GIF — reduce any image file size instantly.

📐

Target Size in KB or MB

Set an exact target file size to resize your photo to a specific KB or MB. The binary search algorithm finds the highest quality that still meets your limit.

✂️

Built-in Crop Before Compress

Crop your image before compressing it — drag handles to select the area to keep, then apply. Combine cropping and compression in one step without any extra app.

🔒

100% Private — No Upload

Unlike TinyPNG, Squoosh or Compressor.io, your images never leave your device. Everything runs locally via the HTML5 Canvas API — zero server upload, zero tracking.
